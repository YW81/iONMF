Description

The modules displayed in the article are stored as part of the Supplementary material of the article:

Strazar M., Zitnik M., Zupan B., Ule J., Curk T.: Orthogonal matrix factorization enables integrative analysis of multiple RNA binding proteins .

Specifically, the low dimensional modules represent the three most related components to positive positions of each RNA binding proteins. They are drawin in Supplementary Section 7 for each data source. Please see the article for further details.

The values represent the z-scores associatied to a particular vector (module), such that they are comparale between modules. Below are example modules for ELAVL1 (PARCLIP) protein.
